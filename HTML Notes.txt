Colors:

    green | blueviolet | red | white

Color Codes: (Given by designer)

1. Hex Codes

2. RGB (Red Green Blue)

3. HSL (Hue Saturation Lightness)

1. Hex Codes: Hexa Decimal Format

    Syntax:

        #000000 - #FFFFFF

            First two values - Red Color

            Second two values - Green Color

            Third two values - Blue Color

    Eg.

        #001E4A

        color: #001E4A;

2. RGB:

    R - Red

    G - Green

    B - Blue

        Syntax:

            rgb(0, 0, 0) - rgb(255, 255, 255);

            rgba(0, 0, 0, 0) - rgba(255, 255, 255, 1);

            a - Opacity

3. HSL

    H - Hue - Specifies the intensity of the color. - (0 - 360)

        0 - 120 - Red

        121 - 240 - Green

        240 - 360 - Blue

    S - Saturation - If saturation is more, color will be more brighter. (0% - 100%)

    L - Lightness - If lightness is more, color will be more lighter. (0% - 100%)

    Syntax:

        color: hsl(0, 0%, 0%);

Positions: Specifies the position of an HTML Element.
    -> Using positions, we can change the position of an HTML Element from one to another.
    Syntax:

            sticky: Element will be fixed to the web page but when user starts scrolling the web page.
            relative: Used to move element from one position to another.
            absolute: Used to move child element from one position to another, by taking the reference of parent element.
