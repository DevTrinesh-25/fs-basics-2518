Spread and Rest Operators:
Spread Operator: Extract values from array or object as individual values.
    Syntax:
        const arr = [value1, value2];
        const arr2 = [value3, ...arr]
                    [value3, value1, value2]
        const arr2 = [...arr, value3]
            [value1, value2, value3]
Rest Operator: Store rest of the values passed to the function definition.
    Syntax:
        function functionName(param1, param2) {
        }
